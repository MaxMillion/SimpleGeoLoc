# SimpleGeoLoc

Simple geolocation functions. Add items with geolocation information, then find items near a point/distance.

## Usage

## Installation

Via npm on Node:

```
npm install simplegeoloc
```

## Usage

Reference in your program:

```js
var sgl = require('simplegeoloc');
```
